# React & Node.js E-Commerce Website

This project teaches you how to build a complete **E-Commerce website** using **React.js**, **Node.js**, **Express.js**, and **MongoDB**. The website includes features like user login, product management, shopping cart, checkout, and order management. 

## What You Will Learn

* Build a responsive website using HTML and CSS.
* Create user interfaces with React.js.
* Develop backend APIs using Node.js and Express.js.
* Store data in MongoDB.
* Manage application state with Redux.
* Implement user authentication using JWT.
* Upload product images.
* Deploy the application online. 

## Main Features

### User Features

* Register a new account.
* Log in securely.
* Browse products.
* View product details.
* Add products to the shopping cart.
* Update or remove cart items.
* Enter shipping information.
* Choose a payment method.
* Place orders.
* View order history.
* Update profile information.

### Admin Features

* Log in as an administrator.
* Add new products.
* Edit existing products.
* Delete products.
* Manage customer orders.
* Mark orders as delivered.
* Upload product images.
* View all products and orders.

## Project Workflow

1. Clone the project from GitHub.
2. Install MongoDB.
3. Install backend dependencies.
4. Start the backend server.
5. Install frontend dependencies.
6. Start the React application.
7. Create an admin account.
8. Log in as Admin.
9. Add products.
10. Test shopping, checkout, and order management.
